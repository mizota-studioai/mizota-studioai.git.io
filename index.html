Web Designer Ryosuke Mizota

広島県在住のWebデザイナー、カメラマン、WEBマーケティング、製品の導入から検証から選定から色々やってる綾介です。オンラインスクールでいい感じにWeb制作について学び、サイトリニューアルを自分で完結できるよう勉強中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHP、WordPressもいつもギリギリのところで出来てます。クラウドサービスの導入とかが得意です。

Camera

Japanise
